#### Phương pháp 1: Giao diện nút bấm ⭐
1. Gửi `/start` để mở menu
2. Chọn "🚀 Lấy mã bypass"
3. Chọn domain cần lấy mã
4. Chọn platform (Facebook/Google)
5. Nhận mã bypass ngay lập tức!

```
/layma [domain] [platform]
```
